# Netlify Environment Variables Setup

Add these environment variables in your Netlify dashboard:
**Site Settings → Build & Deploy → Environment Variables**

## Required Variables

### AI Model APIs
```bash
# Hugging Face API Token (Required)
# Get it for FREE at: https://huggingface.co/settings/tokens
HF_TOKEN=hf_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

# Google Gemini API Key (Optional but recommended)
# Get it at: https://makersuite.google.com/app/apikey
GEMINI_API_KEY=AIxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## How to Get Your Tokens

### Hugging Face Token
1. Go to [https://huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)
2. Click "New token"
3. Give it a name (e.g., "ArchiTech Wize")
4. Select "read" permission
5. Copy the token (starts with `hf_`)

### Google Gemini API Key
1. Go to [https://makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)
2. Click "Create API key"
3. Copy the key (starts with `AI`)

## Adding to Netlify

1. Open your Netlify dashboard
2. Select your site
3. Go to **Site settings**
4. Click **Environment variables** in the sidebar
5. Click **Add a variable**
6. Add each variable:
   - **Key**: `HF_TOKEN`
   - **Value**: Your Hugging Face token
   - **Scopes**: All (Production, Deploy Previews, Branch deploys)
7. Click **Create variable**
8. Repeat for `GEMINI_API_KEY`
9. **Deploy your site** for changes to take effect

## Verification

After deployment, visit `/model-health` on your site to verify:
- ✅ Hugging Face: Connected
- ✅ Gemini: Connected
